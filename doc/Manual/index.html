Synopsis Developer Reference Manual

By running make in the docs/RefManual directory you can generate documentation for Synopsis. Two sections are generated, the Python code and the C++ Parser. The Python code consists of everything except the C++ parser, such as the Core libraries, the formatters, the linker, and the IDL and Python parsers. The C++ parser is written in C++ for speed and to allow interfacing with the Open C++ Compiler (OCC).

You may visit these manuals here:

  • Core, Formatters, Linkers, (some) Parsers.
  • C++ Parser.
